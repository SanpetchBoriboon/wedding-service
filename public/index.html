Wedding Card Online Service

Welcome to the Wedding Card Online Service API! This service helps you create and manage beautiful wedding invitation cards.

Available Endpoints:

GET / - API welcome message
GET /api/cards - Get list of wedding card templates
POST /api/cards - Create a new wedding card
Body: { "title": "Wedding Title", "message": "Your message", "template": "template_name" }
GET /health - Health check endpoint

This is a static HTML page served from the public directory. The API is accessible via the endpoints listed above.

Getting Started:

  1. Make sure the server is running
  2. Use a tool like Postman or curl to interact with the API
  3. Create your first wedding card by sending a POST request to /api/cards
